AI-Based Food Demand Forecasting System to Reduce Food Waste
Project Overview

Food waste is a major challenge in the hospitality and food service industry. One of the main causes of food waste is over-preparation, where kitchens produce more meals than customers actually order. This project explores how machine learning can be used to predict food demand and help reduce unnecessary waste.

The goal of this project is to build a predictive model that forecasts the number of meal orders using historical data. By improving demand predictions, food service providers can prepare more accurate quantities of food while maintaining customer satisfaction.

Objective

The objective of this project is to develop a machine learning model that predicts food demand based on historical order data. The model analyzes factors such as pricing, promotions, and meal categories to estimate future demand.

The predictions generated by the model can support better planning in food service operations and help reduce excess food preparation.

Dataset

The project uses a food demand forecasting dataset composed of three main files:

train.csv – historical food order data

meal_info.csv – information about meal categories and cuisines

center_info.csv – information about fulfillment centers

Important variables used in the analysis include:

week (time index)

center_id (fulfillment center location)

meal_id (meal identifier)

checkout_price (final selling price)

base_price (original price)

emailer_for_promotion (promotion indicator)

homepage_featured (homepage promotion indicator)

num_orders (target variable representing total orders)

The datasets were merged and prepared for machine learning analysis.

Methodology

The project follows a typical machine learning workflow.

First, the datasets were cleaned and merged into a single dataset. Categorical variables such as cuisine and meal category were converted using one-hot encoding so they could be used by the machine learning model.

The dataset was then divided into training and testing sets using an 80/20 split.

A Random Forest Regressor model was used to predict food demand. Random Forest is an ensemble learning method that combines multiple decision trees to produce more stable and accurate predictions.

Model Evaluation

The performance of the model was evaluated using regression metrics.

RMSE: 142.70
R² Score: 0.865

The R² score indicates that the model explains approximately 86.5% of the variation in meal demand, which suggests strong predictive performance. The RMSE value indicates that the average prediction error is approximately 142 orders.

Data Exploration

The dataset was first analyzed to understand the distribution of meal orders. A histogram was used to visualize the distribution of the num_orders variable. The results show that most orders fall within a moderate range, with fewer extremely large orders.

Weekly demand trends were also analyzed by grouping the data by week. The time-series plot shows fluctuations in demand over time, highlighting the need for predictive models to better anticipate demand patterns.

Model Prediction

The Random Forest model was trained using several features, including:

week

center_id

meal_id

checkout_price

base_price

promotional indicators

meal categories

The model learns patterns between these variables and customer ordering behavior to estimate future demand.

Feature Importance

Feature importance analysis was performed to determine which variables most strongly influence the model’s predictions.

The results show that checkout price is the most important feature, followed by operational area, homepage promotions, and meal category variables. This indicates that pricing strategies and promotional visibility significantly influence customer demand.

Prediction Visualization

A scatter plot comparing actual orders and predicted orders shows a strong positive relationship between the two variables. Many data points cluster along the diagonal trend, indicating that the model predictions closely match the actual demand values.

Overall, the model demonstrates strong predictive performance and shows that machine learning can support better food preparation planning and help reduce food waste.
